# INGEBORGER

Ingeborger é uma aplicação desenvolvida em Python utilizando tkinter e PIL para gerenciamento de recursos em um rpg. A aplicação conta com um gerador de encontros aleatórios, dragonário com dados de dragões, um gerador de testes e lista de personagens. O código é modularizado no principal Ingenborger.py, list.py que contem as listas de informações e as pastas img com imagens de dragões e pp com os personagens.

## Instalação

Para instalar o RPG Manager, você precisará ter o Python 3 instalado em sua máquina. Você também precisará instalar o tkinter e o PIL. Para instalar o tkinter, execute o seguinte comando no terminal:

```
pip install tkinter
```

Para instalar o PIL, execute o seguinte comando no terminal:

```
pip install Pillow
```

## Uso

Para usar o RPG Manager, execute o seguinte comando no terminal:

```
python ingenborger.py
```

A aplicação será aberta em uma janela do terminal. Você poderá usar o menu para navegar pelas diferentes funcionalidades da aplicação.
